# school-cronicle


This app create the yearly chronical of a school. 

There will be 
- teachers
- appointment (e.g. school conference, Parents' conference day, Awarding of certificates, ...)
- events (excursions, Course exits, course, workshops, travels)
- images (from teachers, appointments, ...)
- cronicles (consists of teachers, images, chapters, sub-chapters, ...)
